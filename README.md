# Sail2Learn SEO Strategy & Migration

This repository contains the complete SEO, AI Optimization, and Technical Strategy for the sailing instruction business **Sail2Learn** (Sarasota/Bradenton, FL).

## Repository Contents

*   [`sail2learn-seo-strategy.md`](./sail2learn-seo-strategy.md): The master SEO strategy. It prioritizes tasks based on an **Impact-Weighted Roadmap** (Critical vs. Moderate vs. Sentimental impact) and contains the JSON-LD Schema code specifically mapped to the business.
*   [`sail2learn-execution-guide.md`](./sail2learn-execution-guide.md): The step-by-step UI checklist for executing the strategy directly inside Squarespace and Google Business Profile.
*   [`squarespace-seo-workflow.md`](./squarespace-seo-workflow.md): A guide on how to work with an AI assistant in a closed ecosystem like Squarespace, detailing how to find hidden SEO backend settings.
*   [`squarespace-to-code-migration.md`](./squarespace-to-code-migration.md): A strategic breakdown of the pros/cons of migrating off Squarespace to a "Code-First" (Astro/Next.js) architecture for long-term SEO dominance and AI integration.

## About the Project
This strategy is designed to resolve critical Core Web Vitals issues, implement Local SEO dominance (Google Maps Pack), and establish E-E-A-T (Experience, Expertise, Authoritativeness, and Trustworthiness) for AI engines.
